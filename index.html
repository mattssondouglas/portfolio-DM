Beer Exchange

Big data visualization project co-developed with Kaz at Tortuga Coders Full Stack Javascript Development Bootcamp summer 2022. This game tracks prices of drinks in a bar as orders are entered. The price of a given beer goes up and down based on demand and time, with price history displayed on a live chart. Built with Javascript, Express.js, Node.js, Handlebars.js, Chart.js, MongoDB, Axios-HTTP, among other tools.

Have a look

Airbnb

Developed at Tortuga Coders Full Stack Javascript Bootcamp, this big project is a substantial remake of the Airbnb website. Users are able to search for houses using text but also a list of other criteria, such as price or number of rooms, then browse details of each house in an individual page, read other users reviews, submit a review and send a message to the host to request a booking. I have implemented a Signup / Login system, so users are able to create a profile and list houses (become hosts). Built with Javascript, Bootstrap, Express.js, Node.js, Handlebars.js, MongoDB, Axios-HTTP, among other tools.
